Forms in CSS3 & HTML5

How about a form?

CSS3 & HTML5 for forms is partially baked...

The fallback for any non supported input is a text input. So go ahead and use the new fields that work in a few browsers if it makes sense.

  • Unfortunately, the date picker seems to only work in Opera.
  • I love the range, but it is not quite baked either... how many did I choose? Of course, some scripting can fix this one.
  • Number - the increment is cool. It is keyboard accessible as well.
  • Required is cool. If it gets widespread support (IE9?, Safari?).
  • Email, URL, and Phone are awesome on mobile. They pull up the correct keyboard. Go ahead and use them.
  • Opera 11 supports a color picker... so how many people use Opera 11?

Let's make this one look cool at least.
