Start with a content box

Today, we build a content box (with an inner box)

And add modifiers to style it

Modular CSS / OOCSS / Visual Patterns

Our box will be simple, modular, scalable, modifiable and reusable.

The goal is to create reusable chunks of code for repeatable visual patterns.

The CODE for our box

<div class="cB"> <div class="inner modifier classes go here"> <h3 class="hThree">Component Title or Heading</h3> <p>Component Content</p> </div> </div>

We could use the article element as well - even better semantics.

<article> <div class="inner modifier classes go here"> <h3 class="hThree">Component Title or Heading</h3> <p>Component Content</p> </div> </article>

And the basic CSS

.cB {margin:.5em 0; z-index:1001; position:relative;} .inner {margin:12px;} .cB .inner {padding:8px;}
